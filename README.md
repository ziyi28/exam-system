# 智能云考 (Smart Cloud Exam)

本项目是一个面向 IT 教育场景开发的在线考试与智能阅卷平台。系统实现了从**自定义组卷、在线答题到 AI 智能批阅**的全流程闭环，有效解决了主观题阅卷成本高、反馈滞后等核心痛点。

## 🚀 技术选型

* **后端核心**：Java, Spring Boot, **Spring WebFlux**, MyBatis-Plus
* **存储/缓存**：**MySQL**, **Redis** (ZSet 排序)
* **中间件**：**MinIO** 对象存储, **Apache POI** 报表处理
* **AI 能力**：**DeepSeek LLM** (API 异步集成)
* **前端**：Vue 3, TypeScript, ECharts

## 🌟 项目亮点与职责

* **🤖 AI 智能引擎开发**：基于 **Spring WebFlux** 异步集成 DeepSeek API，设计结构化 **Prompt** 引导模型输出。实现了**指数退避重试机制**，确保了 AI 生成题目的可用性与稳定性。
* **⚙️ 复杂业务逻辑设计**：研发动态智能组卷引擎，支持按难度、分类等多维规则随机抽题。利用 **@Transactional** 确保题目、试卷及考试记录在多表交互下的**数据强一致性**。
* **⚡ 高性能中间件应用**：利用 Redis **ZSet** 实现热门题目实时排行榜，采用异步解耦方式优化主查询性能。集成 **MinIO** 封装对象存储，解决大规模附件存储冲突。
* **📊 数据处理优化**：基于 **Apache POI** 实现多格式 Excel 题目批量导入，设计**部分成功策略**避免单条脏数据阻塞流程。

## 📈 项目成果

* **智能判分**：主观题批阅准确率达 **90%** 以上（模拟数据），阅卷效率提升约 **80%**。
* **性能提升**：通过异步化改造，系统核心接口响应耗时降低了 **40%**。
